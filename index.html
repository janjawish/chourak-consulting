Nos expertises

Data Engineer

  • Concevoir et maintenir les pipelines de données (ETL/ELT)
  • Mettre en place l’infrastructure data (Data Lake, Data Warehouse)
  • Optimiser la performance et la qualité des données
  • Automatiser les flux via des orchestrateurs

Clouds

Azure (Data Factory, Synapse, SQL Database), AWS (Glue, Redshift), GCP (BigQuery)

Outils

Databricks, Snowflake, Airflow, DBT — Langages : Python, SQL, Spark

Data Analyst

  • Analyser les données pour en extraire des insights métier
  • Créer des tableaux de bord et indicateurs clés (KPI)
  • Modéliser les données pour la visualisation
  • Collaborer avec les équipes métier pour la prise de décision

Clouds

Azure, AWS, GCP (pour les sources de données)

Langages

SQL, DAX, parfois Python

Data Scientist

  • Construire des modèles prédictifs et statistiques
  • Nettoyer, explorer et préparer les datasets
  • Déployer des modèles de Machine Learning en production
  • Interpréter et communiquer les résultats

Clouds

Azure ML, AWS Sagemaker, GCP Vertex AI

Outils

Databricks, Jupyter, MLflow, Snowflake

Langages

Python (scikit-learn, pandas, TensorFlow, PyTorch), SQL

IA Engineer

  • Développer des solutions IA Générative
  • Concevoir et déployer des assistants IA répondant aux besoins métiers
  • LLM, RAG, intégration de modèles
  • Industrialiser les modèles d’IA / ML
  • Mettre en place des pipelines de training et déploiement (MLOps)
  • Gérer la scalabilité et la performance des modèles
  • Intégrer les modèles dans les applications métiers

Clouds

Azure ML, AWS Sagemaker, GCP Vertex AI

Outils

Databricks, MLflow, Airflow, Docker, Kubernetes, Copilot Studio, AI Foundry

Langages

Python, SQL, parfois Java/Scala
